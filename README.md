### 简介

#### 技术栈

1. Vue全家桶(vue2,vue-router，vuex)
2. axios
3. element-ui
4. (nodjs)网易云音乐接口服务[https://github.com/Binaryify/NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi)
5. 歌词滚动页使用better-scroll以及lyric-parser实现
